# ETH Video Portal Optimizations

A Chrome extension that enhances the [ETH Zurich video portal](https://video.ethz.ch) with faster playback speeds and lecture bookmarks. It also hides the sidebar. 

## Installation

1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer mode** (top-right toggle).
4. Click **Load unpacked** and select the folder containing `manifest.json`.

## Usage

- Navigate to [video.ethz.ch](https://video.ethz.ch).
- Open the speed menu in the video player — **2.5x** and **3x** options will be listed.
- On a lecture archive page, click the bookmark icon (next to the share button) to save it.
- Visit the homepage to see all your bookmarks.
